n 個の品物があり、i 番目の品物のそれぞれ重さと価値が weight[i],value[i]となっている
(i=0,1,...,n−1)。これらの品物から重さの総和が W を超えないように選んだときの、価値の総和の最大値を求めよ。

・１重さあたりの価値を算出し、入力された重量に対して最大量の価値となる品物を選択するアプリを作成しました。
※ただしこのロジックだと、一部 最適解を算出できないケースがありました。
